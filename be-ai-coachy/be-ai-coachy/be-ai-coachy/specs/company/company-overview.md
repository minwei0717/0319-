## 事業服務範疇

### 一、回收報廢服務

從源頭收集各類可回收資源，提供便利的回收管道。

| 服務項目                | 內容說明                                                 |
| ----------------------- | -------------------------------------------------------- |
| **資源回收服務**  | 金屬類、塑膠類、紙類、電子類回收，全台近 20 間回收站     |
| **廢汽機車報廢**  | 線上報價、免費拖吊、車籍代辦、獎勵金申請，最快 24hr 完成 |
| **家電回收**      | 電腦、電視、冰箱、洗衣機、冷氣等大型廢家電到府回收       |
| **zero zero APP** | 回收打卡、家電回收、廢車回收、文件銷毀一指預約           |

**可回收項目**：

- 金屬類：廢鐵、廢銅、廢鋁、鋼筋、白鐵、紅銅、電線電纜
- 塑膠類：單一材質塑膠製品、寶特瓶、牛奶瓶、清潔用品瓶
- 紙類：廢紙、紙箱、書報
- 電子類：廢電腦、電子零件、貴金屬、IC

---

### 二、廢棄物清運服務

為企業提供客製化與統包式的廢棄物處理解決方案。

| 服務項目                 | 內容說明                                              |
| ------------------------ | ----------------------------------------------------- |
| **企業清運服務**   | 工廠、政府機關、學校、銀行之回收物/廢棄物全程管理     |
| **綜合報廢服務**   | 企業設備報廢、產線汰換、辦公設備淘汰                  |
| **生活垃圾清運**   | 社區、大樓、公寓、企業定期清運                        |
| **文件/硬碟銷毀**  | 封閉式機密文件銷毀、硬碟三種銷毀方式（符合 ISO27001） |
| **廢棄物棄置清運** | 處理遭傾倒之大型廢棄物、有害事業廢棄物                |
| **事業廢棄物處理** | 甲級許可證、科技大廠下腳料回收、駐廠管理              |
| **廢木材清運**     | 建築工地、裝潢拆除、工廠淘汰木製品                    |
| **石綿瓦拆除**     | 專業團隊安全拆除，符合環保法規                        |
| **廢棄物減量**     | 源頭減量規劃、分類優化、再利用方案設計                |

**合作客戶**：台積電、聯電、友達、大型連鎖通路、縣市政府

---

### 三、再生循環服務

將回收物轉化為再生材料與產品，實現循環經濟。

| 服務項目                      | 內容說明                                              |
| ----------------------------- | ----------------------------------------------------- |
| **再生塑膠材料（PCR）** | 消費後塑膠再製成 PCR 再生塑膠粒，可減少 65-75% 碳排放 |
| **再生循環產品**        | 環保循環箱、再生名片盒、物流載具、客製化再生商品      |
| **永續循環服務**        | 廢棄物再利用規劃、封閉式循環系統、產品生命週期延續    |
| **環保活動規劃**        | 環保教育活動、淨灘活動、ESG 永續活動、品牌形象活動    |

**PCR 應用領域**：日化品容器、洗沐用品瓶、化妝品包裝、兒童玩具、牙刷柄

**合作品牌**：歐萊德 O'right、淨毒五郎、洲際酒店、日月光

### 四、國際貿易事業

拓展海外市場，進行再生原料進出口貿易。

| 服務項目               | 內容說明                                  |
| ---------------------- | ----------------------------------------- |
| **再生原料出口** | PCR 再生塑膠粒外銷                        |
| **海外據點營運** | 泰國據點（2020 年設立）                   |
| **國際認證取得** | 德國藍天使、美國 FDA NOL、GRS、TUV 等認證 |

**國際認證**：

- 德國藍天使 EuCertPlast 認證（亞洲第一家）
- 美國 FDA NOL 授權（全台首家食品接觸再生材料）
- GRS 全球回收標準認證
- TUV-ISO 14067 碳足跡認證
